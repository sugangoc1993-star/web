# Country Restaurant Website

A beautiful, modern website for a country restaurant featuring all essential sections and functionality.

## Features

- **Header**: Logo (left), Navigation menu (center), Search box (right)
- **Image Slider**: Auto-rotating banner with restaurant photos
- **Introduction Section**: Restaurant story with text and photo
- **Menu Section**: Dishes with prices and photos
- **Space Gallery**: Showcase of restaurant design and atmosphere
- **Customer Reviews**: Multiple reviews with ratings
- **Footer**: Contact information, opening hours, and social media links (Facebook, Zalo)

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with responsive design
- `script.js` - Interactive functionality (slider, search, animations)

## Customization

### Logo Image
Replace the placeholder logo in the header:
```html
<img src="path/to/your/logo.png" alt="Restaurant Logo" id="logo-img">
```

Currently using: `https://via.placeholder.com/150x60/8B4513/FFFFFF?text=Country+Restaurant`

### Images
The website uses placeholder images from Unsplash. Replace them with your own restaurant photos:
- Slider images
- Introduction section photo
- Menu item photos
- Space gallery photos

### Content
- Update restaurant name, description, and story
- Modify menu items, prices, and descriptions
- Add real customer reviews
- Update contact information and opening hours
- Update social media links

### Colors
Main color scheme uses brown (#8B4513) for a country restaurant theme. Edit colors in `styles.css`:
- Primary color: `#8B4513`
- Adjust as needed for your brand

## How to Use

1. Open `index.html` in your web browser
2. Or use a local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve`
   - VS Code: Use the "Live Server" extension
